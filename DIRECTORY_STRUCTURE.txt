🎨 Frontend Architecture
═══════════════════════════════════════════════════════════════

React 18 + Vite
├── Routing: React Router v6
├── Styling: Tailwind CSS
├── Animations: Framer Motion
├── Charts: Recharts
├── Maps: React Leaflet
├── HTTP: Axios
└── Icons: Lucide React

Pages Flow:
Landing → Upload → Dashboard → [Anomalies, Predictive, Map, Maintenance]
                              ↓
                         Asset Detail

🔧 Backend Architecture
═══════════════════════════════════════════════════════════════

FastAPI + Python
├── Server: Uvicorn ASGI
├── Data: Pandas + NumPy
├── ML: Scikit-learn
├── Files: Python Multipart
└── CORS: Enabled

API Structure:
/                           → Health check
/api/upload                 → File upload
/api/dashboard/overview     → Dashboard data
/api/anomalies             → Anomaly list
/api/assets/{id}           → Asset details
/api/predictive/risk       → Predictions
/api/map/data              → Map data
/api/maintenance/plan      → Planning

📊 Data Flow
═══════════════════════════════════════════════════════════════

1. User uploads CSV → UploadPage.jsx
2. Files sent to → backend/main.py (/api/upload)
3. Backend processes → Pandas + NumPy
4. Risk calculated → Custom algorithm
5. Data stored → In-memory (demo)
6. Frontend fetches → api.js (Axios)
7. Components render → React pages
8. Charts display → Recharts
9. Map shows → Leaflet
10. User interacts → Smooth animations
